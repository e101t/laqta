# سياسة الخصوصية - LAQTA

نجمع بيانات الحساب مثل الاسم، الهاتف أو البريد، المدينة، الدور، صور الملف، محتوى المستخدم، الرسائل، الحجوزات، بيانات النشاط، ومعرّفات الجهاز اللازمة للأمان.

تُستخدم البيانات لتشغيل المنصة، تحسين الاكتشاف والحجوزات، إرسال إشعارات Firebase FCM، تحليل الأداء، كشف الاحتيال، وحماية المجتمع.

الصور والفيديوهات تُخزن عبر backend وMinIO. بيانات الدفع تُعالج عبر Stripe ولا نخزن أرقام البطاقات داخل التطبيق.

لا نبيع بيانات المستخدمين. قد نشارك بيانات محدودة مع أطراف تشغيلية مثل Stripe أو Firebase أو جهات قانونية عند الحاجة.

يمكن طلب حذف الحساب من التطبيق أو عبر support@laqta.cloud. قد نحتفظ بسجلات محدودة لأسباب قانونية أو أمنية أو نزاعات مفتوحة.
