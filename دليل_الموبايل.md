# 📱 دليل تشغيل LAQTA على الهاتف المحمول

## ✅ التطبيق جاهز كـ PWA (Progressive Web App)!

### 🚀 طرق التشغيل على الهاتف:

#### الطريقة الأولى: تشغيل سريع (الأسهل)
1. **انقر على**: `تشغيل_على_الموبايل.bat`
2. **اتبع التعليمات** التي ستظهر على الشاشة
3. **على هاتفك**، اذهب إلى العنوان المعروض

#### الطريقة الثانية: خطوة بخطوة
1. **افتح Command Prompt**
2. **انتقل إلى المجلد**: `cd build\web`
3. **شغل السيرفر**: `python -m http.server 8080`
4. **على هاتفك**، اذهب إلى: `http://[عنوان-الكمبيوتر]:8080`

### 🔍 لمعرفة عنوان الكمبيوتر:
```cmd
ipconfig
```
أو
```cmd
hostname
```

### 📲 تثبيت التطبيق على الهاتف:
1. **افتح التطبيق** في متصفح الهاتف
2. **اضغط على زر القائمة** (⋮) في المتصفح
3. **اختر**: "إضافة إلى الشاشة الرئيسية" أو "تثبيت التطبيق"
4. **سيظهر أيقونة LAQTA** على الشاشة الرئيسية!

### ✨ مميزات PWA:
- ✅ **يعمل كتطبيق أصلي** على الهاتف
- ✅ **يمكن تثبيته** على الشاشة الرئيسية
- ✅ **يعمل بدون إنترنت** (بعد التحميل الأول)
- ✅ **إشعارات فورية**
- ✅ **واجهة محسنة** للهاتف المحمول

### 🔑 بيانات تسجيل الدخول:
```
رقم الهاتف: +201234567890
كود OTP: 123456
```

### 🛠️ إذا لم تعمل الطريقة الأولى:
1. تأكد من **تثبيت Python** على الكمبيوتر
2. تأكد من **نفس الشبكة** (WiFi) بين الكمبيوتر والهاتف
3. تأكد من **عدم وجود防火墙** يحجب المنفذ 8080
4. جرب **منفذ آخر**: `python -m http.server 3000`

### 📞 الدعم:
إذا واجهت أي مشكلة، تأكد من:
- الكمبيوتر والهاتف على نفس الشبكة
- تشغيل Command Prompt كمدير (Run as Administrator)
- عدم وجود برامج أمان تحجب الاتصال
